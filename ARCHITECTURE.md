## Performance Considerations

### Model Loading
- **When**: Once at application startup
- **Where**: Global variables in model_loader.py
- **Memory**: ~100-500MB depending on models
- **Time**: 2-10 seconds on first load

### Prediction Speed
- **EfficientNet**: ~100-500ms per image (CPU)
- **EfficientNet**: ~10-50ms per image (GPU)
- **XGBoost**: ~5-20ms per prediction
- **Combined**: Sum of both

### Optimization
- Models kept in memory (no reload per request)
- Batch processing possible for multiple images
- GPU acceleration if CUDA available
- Image preprocessing optimized
